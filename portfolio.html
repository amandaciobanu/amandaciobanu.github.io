Shades and Colors Generator

Shade and Colors

Shades And Colors Generator

November 2020

React is a great and powerful front-end framework. My current object is to master it. This project was focused on solving a personal problem I sometimes have using other similar tools. On choosing the color shades for my projects I had difficulties in finding an application that would give me plenty of options side by side and compare them.

This application uses npm Value.js package to call hex colors values and percentage weight, a function to convert RGB values to hex, and writeText javascript function to copy the colors to the clipboard.
